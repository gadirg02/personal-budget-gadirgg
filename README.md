# Личный бюджет

UI-polish релиз.

Изменения:
- на мобильной версии добавлено нижнее меню;
- карточки и таблицы стали компактнее;
- суммы визуально выделены крупнее;
- служебные подписи стали меньше;
- добавлены/усилены цветовые метки для статусов, просрочек и резерва;
- настройки сделаны компактнее через раскрывающиеся блоки;
- на ПК ограничена ширина контента;
- кнопки удаления сделаны менее заметными, чтобы снизить риск случайного нажатия.

Финансовая логика, Supabase, автологин и автосинхронизация не менялись.
