# Личный бюджет

Версия с будущими доходами.

Изменения:
- доходы с будущей датой отображаются как «Ожидается»;
- будущие доходы не участвуют в остатках до наступления даты;
- доходы без даты считаются полученными для совместимости со старыми данными;
- на главной добавлен блок «Ближайшие поступления»;
- в сводке добавлены «Доходы получены», «Ожидается доходов» и прогноз с будущими доходами.

Данные сохраняются локально и синхронизируются с Supabase при подключенном аккаунте.
