# Личный бюджет

Откройте `index.html` в браузере или разместите файлы на GitHub Pages.

## Что есть
- Локальное сохранение в браузере.
- Облачный режим через Supabase.
- Автосинхронизация: после входа изменения автоматически сохраняются в Supabase через несколько секунд.
- Ручные кнопки «Загрузить из облака» и «Сохранить в облако» оставлены как резерв.
- Настройки разделены на поднастройки: база данных, обслуживание, месяцы/архив, категории расходов.

## Важно
Для облачного режима в сайт вставляется только Project URL и Publishable/anon key. Secret/service_role key сюда вставлять нельзя.


## Автоматическая синхронизация

В этой версии ручные кнопки больше не нужны для обычной работы.

1. Один раз настройте Supabase в `cloud-config.js`:
   - `url` — Project URL из Supabase;
   - `key` — Publishable / anon key.
2. Загрузите файлы на GitHub Pages.
3. На каждом устройстве один раз войдите по email/паролю.
4. Дальше сайт сам загружает данные при открытии и сохраняет изменения в Supabase через несколько секунд.

В интерфейсе ручные кнопки оставлены только как аварийный вариант: «Обновить из облака» и технические настройки.
