# Личный бюджет

Откройте `index.html` в браузере или выложите папку на GitHub Pages.

В этой версии настройки разделены на поднастройки:
- База данных — импорт, экспорт, Supabase и вход в облако;
- Обслуживание — очистка сумм и пустой шаблон;
- Месяцы и архив — копирование месяца, архивирование и разархивирование;
- Категории расходов — компактное управление категориями расходов.

Данные могут храниться локально в браузере или в Supabase, если включить облачный режим.
